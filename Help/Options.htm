EtaComp - Etalonnage de comparateurs

Options du programme

Cette page décrit les fonctionnalités proposées par le fenêtre de paramétrage des options du programme. La fenêtre des options est constituée de 4 onglets.

Paramétrage général:

  • Emettre un beep après chaque mesure:
    Lorsque cette option est activée, l'ordinateur émet un "beep" après chaque pression sur le bouton poussoir du comparateur numérique.
    Cette option permet d'éviter à l'opérateur de regarder l'écran de l'ordinateur après chaque mesure pour vérifier que celle-ci a effectivement été prise en compte par le programme.
  • Demander confirmation avant d'arrêter le programme:
    L'activation de cette option permet d'afficher une boîte de dialogue demandant confirmation avant d'arrêter le programme.
  • Afficher les astuces du jour au démarrage:
    L'activation de cette option permet d'afficher une fenêtre présentant des informations et des conseils d'utilisation à propos du logiciel.
    Pour afficher les astuces en cours d'utilisation, sélectionner le sous-menu "Astuces du jour" du menu "?" de la fenêtre principale.

Paramétrage de la communication:

Par défaut, le paramétrage de la liaison série RS232 est adaptée au fonctionnement du comparateur numérique utilisé avec le banc de mesure.
La possibilité de modifier le port utilisé et les paramètres de transmission est cependant proposée à l'utilisateur afin de s'adapter à l'éventuel achat d'un nouvel appareil de mesure.

Seuls les utilisateurs avertis devraient modifier les paramètres de communication de l'application. Le choix des paramètres à adopter doit inpérativement se référer à la documentation technique du matériel éditée par le constructeur.

  • La liste déroulante "Port série" permet de sélectionner le port utilisé pour relier le comparateur à l'ordinateur.
    Concrètement, ce paramètre se matérialise par une broche à 9 points située à l'arrière de l'unité centrale de l'ordinateur.
  • La liste déroulante "Vitesse de transmission" permet de synchroniser l'ordinateur avec le flux de données envoyé par le comparateur numérique.
    Si la valeur choisie n'est pas compatible avec le comparateur numérique, EtaComp peut ne plus être capable de décoder les flux de données émis.
  • La liste déroulante "Valeur de parité" permet de fixer le type de contrôle d'erreur logiciel à appliquer sur les paquets de données reçus.
  • La liste déroulante "Bits de données valides" fixe le nombre de bits par paquet de données utilisés pour coder les données effectives.
  • La liste déroulante "Bits d'arrêt" permet de fixer le nombre de bits de fin de paquet de données.
  • Le bouton "Appliquer les paramètres maintenant" permet de tester les valeurs choisies.
    EtaComp indique alors si l'ouverture du port série avec les nouveaux paramètres est possible, sans pour autant garantir que les données reçues seront exploitables. Si une erreur survient, EtaComp rétablie les derniers paramètres valides utilisés pour ouvrir le port série.

Paramétrage des données fixes imprimées dans les procès verbaux:

L'édition d'un procès verbal comporte quelques champs fixes dont le contenu est toujours le même. EtaComp permet d'enregistrer des valeurs par défaut pour ce type de champ et les utilise de manière automatique lors de l'édition d'un procès verbal.

Paramétrage de la mise en page des éditions:

Le nombre de polices de caractères disponibles sur un ordinateur est variable et fonction des différents logiciels installés.
EtaComp permet de choisir la police utilisée lors de l'édition d'un procès verbal ou lors de la visualisation de l'aperçu avant impression.

  • La zone de liste "Police" permet de sélectionner la police à utiliser.
  • Le curseur "Marge horizontale" permet de paramétrer une valeur de retrait à gauche et à droite à appliquer lors des impressions.
    Cette fonctionnalité a été développée afin de permettre le stockage des procès verbaux dans un classeur sans perdre d'information à l'endroit où sont créés les trous de perforation dans la feuille.
