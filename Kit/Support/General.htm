EtaComp - Etalonnage de comparateurs

Présentation générale du programme

EtaComp est un logiciel permettant d'assister un opérateur lors de l'évaluation d'un comparateur mécanique pour déterminer s'il est encore assez fiable pour être utilisé ou s'il doit être rebuté.

Le test d'un comparateur mécanique nécessite l'utilisation d'un autre comparateur, plus précis, qu'il est possible de connecter à un ordinateur afin d'en récupérer les mesures.

EtaComp assure la gestion de toutes les mesures du comparateur de précision envoyées vers l'ordinateur.
